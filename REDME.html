ARROW SYNTAX FROM ES5 TO ES6

  • //ES5 function sum(a, b) { return a + b }
  • //ES6 Arrow function let = sum2 (a, b) => a + b
  • //ES5 function isPositive(number) { return number >= 0 }
  • //ES6 Arrow function let = isPositive2 = number => number >= 0
  • //ES5 function randomNumber() { return Math.random }
  • //ES6 Arrow function let = randomNumber2 = () => Math.random
  • //ES5 document.addEventListener('click', function() { console.log('Click') })
  • //ES6 Arrow function document.addEventListener('click', () => console.log('Click'))
